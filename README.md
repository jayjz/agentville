# AgentVille 1996

A high-signal AI orchestration portfolio piece built with Next.js 16, TypeScript, and 90s-retro aesthetics.

## Stack
- Next.js 16 + TypeScript
- Tailwind CSS
- Framer Motion
- Pixi.js for sprite rendering
- Canvas Confetti

## Features
- Windows 95 desktop shell with draggable windows
- NPC system with state management
- Retro CRT scanline effects
- Pixel-perfect styling
